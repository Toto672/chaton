🎯 Démonstration de l'ellipsis pour les titres de conversation

Cette page démontre comment les titres de conversation longs sont maintenant tronqués avec des points de suspension (...) lorsque l'espace est limité dans la topbar.

📝 Cas d'utilisation
L'ellipsis s'active automatiquement lorsque le titre dépasse la largeur disponible. Le badge de mémoire et les actions restent toujours visibles.
Exemple 1 : Titre court
Le titre s'affiche normalement sans troncature. Aucune ellipsis n'est nécessaire.

💡 Fonctionnalités clés
  • Ellipsis automatique : Le texte est tronqué avec "..." lorsque nécessaire
  • Layout flexible : Le titre se réduit progressivement sans casser le layout
  • Badge visible : Le badge de mémoire reste toujours visible
  • Actions accessibles : Les boutons d'action restent toujours accessibles
  • Responsive : Fonctionne sur toutes les largeurs d'écran
  • Mode sombre : Compatible avec le thème sombre
  • Accessibilité : La zone de glisser-déposer est préservée
🔧 Implémentation technique
CSS clé utilisé : 
.topbar-left {
  display: flex;
  flex: 1;
  min-width: 0; /* Permet au conteneur de se réduire en dessous de sa largeur intrinsèque */
  gap: 8px;
}

.topbar-title {
  white-space: nowrap; /* Empêche le retour à la ligne */
  overflow: hidden; /* Cache le texte qui dépasse */
  text-overflow: ellipsis; /* Affiche "..." pour le texte caché */
  flex: 1;
  min-width: 0; /* Permet au titre de se réduire */
  max-width: 100%;
}
